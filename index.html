üìñ About This Project

This project provides a JavaScript-based converter that parses XSL-FO (XSL Formatting Objects) documents and converts them to PDFMake document definitions. Built using Test-Driven Development (TDD) principles with vanilla JavaScript - no external testing frameworks required.

Current Implementation: Extracting top-level PDFMake parameters (pageSize and pageMargins) from XSL-FO page master definitions.

üíª Code Example

Here's how to use the XSL-FO to PDFMake converter in your code:

// Create converter instance const converter = new XSLToPDFMakeConverter(); // Load your XSL-FO document const xslfoXml = ` <fo:root xmlns:fo="http://www.w3.org/1999/XSL/Format"> <fo:layout-master-set> <fo:simple-page-master master-name="MyPage" page-width="8.5in" page-height="11in" margin="1in"> <fo:region-body/> </fo:simple-page-master> </fo:layout-master-set> </fo:root>`; // Convert to PDFMake definition const pdfMakeDefinition = converter.convertToPDFMake(xslfoXml); // Result: { pageSize: 'LETTER', // Detected as standard US Letter pageMargins: [72, 72, 72, 72], // 1in = 72pt on all sides content: [], _metadata: { ... } }
